Script Generation
The scripts are cleaned up through an automated process. The cleaning process is not perfect and there are likely weird errors, although I have tried to minimize these.
Games are partitioned according to the text source. So, for example, Final Fantasy 7's script has a few long pages whereas Final Fantasy 8's script is divided across many pages, but each page is short.
The code to generate all of the script files is available on github.
This website was written in django and is also available on github.
